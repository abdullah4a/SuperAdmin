## Codes

After cloning the project..
run the following commands:

```
npm install
 npm install --save vue vue-class-component
 npm install --save-dev @babel/plugin-proposal-decorators @babel/plugin-proposal-class-properties
npm install axios

```
and the finally:
```
npm run serve
```
